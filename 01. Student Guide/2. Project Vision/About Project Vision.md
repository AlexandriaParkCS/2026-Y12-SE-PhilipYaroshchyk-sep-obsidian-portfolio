The project vision acts as a catalyst for a software project. It outlines an opportunity that can be addressed by a software project for a clearly identified stakeholders. It explains why the opportunity is important, how new software solution will address it and the required technology and works to realise the software solution.

Project visions come in many forms from ideas captured on the 'back of a napkin' to formal project documents which undergo multiple reviews by experienced project team members. Today, 'pitch decks' are favourite for  contemporary project visions.
