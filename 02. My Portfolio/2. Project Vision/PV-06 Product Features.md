### Core Features

* **Secure user access**: Access limited to authenticated users
* **Pet Owners Onboarding**: Secure profiles which include their personal details verification, their pet type and care needs and location. 
* **Pet Sitters Onboarding**: Secure profiles which include their experience, personal details verification, clear rate information and location.
* **Fair Rating System**: Trusted rating system applied to both pet owners and pet sitters to help both parties, which includes:
	* Star-based rating.
	* Detailed reviews.
* **Search**: 
	* Ability to search and sort qualified pet sitters in your area based on their experience, personal rating, and service cost.
	* Ability to search and sort pet sitting opportunities based on their location, animal type and owner's personal rating. 
* **Problem Reporting and Feedback**: Mechanism that allows pet owners or sitters to report inappropriate behaviour or any other problems encountered during the owner-sitter business engagement.  

## Supporting Features

- **Responsive Interface:** Works across desktop and mobile devices.
- **Secure PII Storage:** All personal identifiable user data stored in a protected SQL database.
- **Secure Profiles**: All user profiles are protected by the application, only intended information is publicly available. Private details (documents, exact address etc)are to be used for the personal verification only and never to be shared with anyone.  

## Constraints

- The system must use secure credentials storage.
- The system must store private information securely.
- The application interface must remain simple and intuitive.
- The application must operate within the capabilities of standard web browsers.

## Assumptions

- Users have basic digital literacy and access to an internet‑enabled device.
- Pet owners will use the application to engage with the services of qualified pet sitters.
- The system will be deployed in an educational or personal‑use context, not for enterprise‑level business.
