***YourCatsAndDogs is a web application that requires no installation. Below is the list of requirements to be able to use the app:***

- An active internet connection to reach the site and load pages.
- A modern, up-to-date web browser with JavaScript and cookies enabled (e.g. Chrome, Firefox, Edge, or Safari) — session login and form submissions rely on this.
- A device with a screen large enough to comfortably view and fill in forms (desktop, tablet, or mobile).
- A valid email address, used to register an account.
- If adding a profile avatar, access to an image file on the device being used.
