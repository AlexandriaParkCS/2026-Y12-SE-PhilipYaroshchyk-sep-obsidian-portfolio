***New User Sign Up Experience

Users can access application's main page only. New users are required to sign up by providing their name, email, password and the type of the **default** account. User name and email are limited to 50 characters. Password requires an upper case, a lower case, a digital and a special character and must be between 8 and 50 characters. Password is masked to prioritise the security.

![[signup_pet_owner.drawio.png]]
![[signup_pet_sitter.drawio.png]]
Upon successful registration, user is redirected to their selected default profile page.  


At this point the user account setup is not complete yet. Links are provided to encourage user to start using the app by entering the information required to complete their profile. 

**Default** account type indicates the type of profile the user must create by providing relevant information. Users can have:
- Owner Profile
- Pet Sitter Profile
- Both Profiles (1 of those is default)

***Existing User Login***

Users that have existing accounts can login by providing their user name and password. If user exists and password is correct, user is logged in and is redirected to their profile page.
