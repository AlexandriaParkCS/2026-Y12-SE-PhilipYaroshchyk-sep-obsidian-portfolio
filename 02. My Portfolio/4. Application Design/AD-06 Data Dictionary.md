**Data dictionary used in YourCatsAndDogs app** 

***users***

| Attribute     | Data Type | Bytes | Description                                                         | Example                                       |
| ------------- | --------- | ----- | ------------------------------------------------------------------- | --------------------------------------------- |
| username      | String    | 256   | Unique user name provided at registration                           | johnd                                         |
| email         | String    | 320   | A valid and unique email address                                    | user@address.com                              |
| postcode      | Integer   | 4     | Postcode of the user's location, used to find nearby pet sitters    | 3121                                          |
| photo         | String    | 512   | File path to the user's uploaded profile photo                      | uploads/photo_a1b2.jpg                        |
| about         | String    | 1000  | Free-text bio describing the user                                   | "Experienced dog walker, available weekends." |
| password_hash | String    | 255   | Salted scrypt hash of the user's password, generated by the library | scrypt:32768:8:1$yhuNz...                     |

***pets***

| Attribute | Data Type              | Bytes | Description                                              | Example  |
| --------- | ----------------------- | ----- | ---------------------------------------------------------| ---------|
| name      | String                  | 100   | Name of the pet                                           | Biscuit  |
| species   | String                  | 50    | Species of the pet                                        | Dog      |
| breed     | String                  | 100   | Breed of the pet                                           | Labrador |
| yob       | Integer                 | 4     | Year the pet was born, used to estimate its age            | 2021     |
| owner_id  | Integer (FK → users)    | 4     | The user who owns this pet                                 | 7        |

***care_types***

| Attribute   | Data Type            | Bytes | Description                                              | Example                                       |
| ----------- | ---------------------- | ----- | ----------------------------------------------------------| -----------------------------------------------|
| name        | String                 | 100   | Name of the type of care required                          | Evening feed                                   |
| description | String                 | 500   | Further detail on what the care involves                   | "One scoop of dry food and fresh water."       |
| schedule    | String                 | 200   | How often/when the care is needed                          | Daily at 6pm                                   |
| pet_id      | Integer (FK → pets)    | 4     | The pet this care type applies to                          | 12                                             |

***bookings***

| Attribute   | Data Type                       | Bytes | Description                                                                          | Example    |
| ----------- | --------------------------------- | ----- | ----------------------------------------------------------------------------------- | -----------|
| pet_id      | Integer (FK → pets)               | 4     | The pet being booked in for sitting                                                   | 12         |
| sitter_id   | Integer (FK → users), nullable    | 4     | The user confirmed as sitter; unset until a booking request is accepted               | 9          |
| start_date  | String                            | 10    | Date the booking starts (ISO 8601)                                                     | 2026-08-01 |
| end_date    | String                            | 10    | Date the booking ends (ISO 8601)                                                       | 2026-08-05 |
| daily_price | Integer                           | 4     | Price charged per day of sitting, in whole currency units                              | 15         |

***reviews***

| Attribute   | Data Type              | Bytes | Description                                                          | Example                              |
| ----------- | ------------------------ | ----- | -----------------------------------------------------------------------| ---------------------------------------|
| booking_id  | Integer (FK → bookings)  | 4     | The booking this review relates to                                     | 4                                     |
| reviewer_id | Integer (FK → users)     | 4     | The user who wrote the review                                          | 7                                     |
| reviewee_id | Integer (FK → users)     | 4     | The user being reviewed                                                | 9                                     |
| score       | Integer                  | 4     | Rating from 1 to 5, enforced by a CHECK constraint                     | 5                                     |
| comment     | String                   | 500   | Optional free-text feedback                                            | "Great with my cat, very reliable."   |

***booking_requests***

| Attribute  | Data Type                | Bytes | Description                                                                        | Example |
| ---------- | --------------------------- | ----- | -----------------------------------------------------------------------------------| --------|
| booking_id | Integer (FK → bookings)     | 4     | The booking a sitter is applying for                                                | 4       |
| sitter_id  | Integer (FK → users)        | 4     | The user applying to sit; unique per booking, enforced by a UNIQUE constraint       | 9       |

***messages***

| Attribute    | Data Type             | Bytes | Description                                                | Example                                  |
| ------------ | ------------------------ | ----- | -------------------------------------------------------------| --------------------------------------------|
| sender_id    | Integer (FK → users)     | 4     | The user who sent the message                                 | 7                                            |
| recipient_id | Integer (FK → users)     | 4     | The user who received the message                             | 9                                            |
| body         | String                   | 2000  | Content of the message                                        | "Hi, is Biscuit good with other dogs?"       |
| created_at   | Timestamp                | 8     | Date and time the message was sent, defaults to the current time | 2026-07-18 14:32:00                       |
