**YourCatsAndDogs** is proposed to be Web Application written using the Python Flask technology stack. It will support 2 user groups:
- Pet Owners
- Pet Sitters

Users of the app will be able to:
* Securely create new accounts
* Securely login and log out 
* Create user profiles of their user group type (Owners or Sitters)
* Pet Sitters will be able to advertise their services
* Pet Owners will be able to search for sitters that satisfy their criteria (experience, location, rating)
* Both groups will be able to write trusted (verified) reviews

***Key Features*** of **YourCatsAndDogs** are:
- Functional
	- User onboarding, authentication, profile creation, advertising of services, search and rating/reviewing
- Performance
	- Responsive user interface, fast application load times, and offline access via service workers
- Security
	- Security by design approach by having session management, protection from SQL injection, password hashing, input validation minimising the user of Personal Identifiable Information (PII)
- Privacy
	- Compliance with the Australian Privacy Act 1988

***User experience*** 
**YourCatsAndDogs** will have a simple and intuitive user interface available on web and mobile. Users will be able to interact with application's modern visual components including forms, charts, tables and maps
