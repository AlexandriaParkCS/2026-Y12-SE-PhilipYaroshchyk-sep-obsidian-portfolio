
**Test Case ID:** TC-012

**Description:** Verify that the user (sitter) can find a booking 

**Preconditions:**
- The user has logged in to the app
- There is a booking request created for the pet species "cat", [see](obsidian://open?vault=2026-Y12-SE-PhilipYaroshchyk-sep-obsidian-portfolio&file=02.%20My%20Portfolio%2F6.%20Application%20Testing%2FAT-TC-011%20-%20User%20is%20able%20to%20create%20a%20booking%20request) 

**Steps:**
1. After login, navigate to the search page by clicking "Find pets to sit" in the menu
2. Fill in the search criteria in the form
3. Click "Search"
 
**Test Data:** 
- Search pets by name, species or breed: "cat"

**Expected Result:** 
- Search returns booking request created in the previous [step](obsidian://open?vault=2026-Y12-SE-PhilipYaroshchyk-sep-obsidian-portfolio&file=02.%20My%20Portfolio%2F6.%20Application%20Testing%2FAT-TC-011%20-%20User%20is%20able%20to%20create%20a%20booking%20request).
