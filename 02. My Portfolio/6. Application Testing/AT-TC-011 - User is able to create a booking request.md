
**Test Case ID:** TC-011

**Description:** Verify that the user can create a new booking request for their pet

**Preconditions:**
- The user has logged in to the app
- The user has onboarded their pet Tommy into the app

**Steps:**
1. After login, navigate to the profile page by clicking "Home" in the menu
2. Scroll down to "You pets section"
3. Click on "Details" against Tommy's entry
4. Fill in the "Create a new booking request" form
5. Click "Request booking" to submit the form
 
**Test Data:** 
- start date: 01/09/2026
- end date: 15/09/2026
- daily price: 50

**Expected Result:** 
- New booking entry has been created and is shown on Tommy's page
